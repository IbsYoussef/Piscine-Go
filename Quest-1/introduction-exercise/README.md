## Problem Statement

After configuring your credentials and creating/cloning the 'piscine-go' repo, create a script named `hello.sh` that prints `Hello <username>!` where `<username>` is your own username.

## Expected Output

```
Hello iyoussef!
```

(Replace `iyoussef` with your actual username)

## How to Work on This

1. Write your solution in `student/hello.sh`
2. Run `./test.sh` to check if it works
3. Run `./run.sh` to see your output
4. Compare with `solutions/hello.sh` if you need help
