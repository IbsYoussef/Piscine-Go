# Quest 1: Shell Scripting Fundamentals

## What You'll Learn

Master shell scripting and command-line tools for text processing, API integration, and problem-solving.

- **Text Processing**: sed, grep, awk
- **API Integration**: curl, jq, JSON parsing
- **Data Manipulation**: Piping, filtering, set operations
- **Problem Solving**: Logic puzzles with CLI tools
